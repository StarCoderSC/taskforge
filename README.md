# 🌱 TaskForge

A cozy productivity companion built with HTML, CSS and JavaScript.

## 🚀 Live Demo

## ✨ Features

- 📝 Add tasks
- ✏️ Edit tasks
- ✅ Mark tasks as completed
- 🗑️ Delete Tasks
- ♻️ Restore deleted tasks
- 🗑️ Delete tasks permanently
- ☢️ Empty recycle bin
- 🔎 Search tasks
- 📊 Statistics panel
- ⛅ Dynamic greetings
- 💭 Random motivational quotes
- 📆 Date and time panel
- ☰ Sidebar navigation
- 🌱 Cozy empty states
- 💾 LocalStorage support

## 🛠️ Built with

- HTML
- CSS
- JavaScript
- LocalStorage

## 🌱 About

TaskForge is a cozy productivity companion designed to help organize daily tasks in a simple and enjoyable way.

The project started as a basic todo list and gradually evolved into a future-rich application with search, statistics, a recycle bin, dynamic, greetings, quotes, and more.
